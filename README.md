# Portfólio para GitHub Pages

## Estrutura
- `index.html`: página inicial
- `styles.css`: estilos gerais
- `assets/fundo-cordel.jpg`: imagem de fundo da capa
- `jornalismo.html`
- `design.html`
- `fotografia.html`
- `edicao-video.html`

## Como usar
1. Envie todos os arquivos para seu repositório.
2. Coloque sua imagem de fundo em `assets/fundo-cordel.jpg`.
3. Edite os textos do `index.html`.
4. Troque os links `https://seu-link-aqui.com` pelos seus links reais.
5. Ative o GitHub Pages nas configurações do repositório.

## Onde trocar as cores
No arquivo `styles.css`, altere:
- `--color-jornalismo`
- `--color-design`
- `--color-fotografia`
- `--color-video`

## Onde trocar o título grande
No `index.html`, altere o conteúdo da tag `<h1>`.
